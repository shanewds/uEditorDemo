# uEditorDemo
百度富文本编辑器的完美整合到SSM框架中


第一步:从百度富文本编辑器官网下载源码,找到JSP文件下的lib文件夹,把jar添加上;<br>
第二步:拷贝需要加载的js,css等,放到指定的目录下;<br>
第三步:拷贝官方提供的java代码到指定的包下即可;<br>


#本项目参考其它的资料,做了两个富文本编辑器样式的显示
